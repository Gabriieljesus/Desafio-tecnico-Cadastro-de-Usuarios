# 👤 Cadastro de Usuários

Aplicação Full Stack de cadastro e gerenciamento de usuários, desenvolvida como desafio técnico. Permite cadastrar, listar e remover usuários de forma simples e intuitiva.

---

## 🚀 Tecnologias

**Frontend**
- React
- Vite

**Backend**
- Node.js
- Express

---

## ✨ Funcionalidades

- ✅ Cadastrar um novo usuário
- ✅ Listar todos os usuários cadastrados
- ✅ Remover um usuário

## ⚙️ Como rodar o projeto

### Pré-requisitos

- Node.js instalado
- npm ou yarn

### Frontend

```bash
# Clone o repositório
git clone https://github.com/Gabriieljesus/Desafio-tecnico-Cadastro-de-Usuarios.git

# Acesse a pasta do projeto
cd Desafio-tecnico-Cadastro-de-Usuarios/Cadastro-de-Usuarios

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```

### Backend

```bash
# Acesse a pasta do backend
cd backend

# Instale as dependências
npm install

# Rode o servidor
npm start
```

## 📡 Endpoints da API

| Método | Rota | Descrição |
|--------|------|-----------|
| GET | `/users` | Lista todos os usuários |
| POST | `/users` | Cadastra um novo usuário |
| DELETE | `/users/:id` | Remove um usuário |
